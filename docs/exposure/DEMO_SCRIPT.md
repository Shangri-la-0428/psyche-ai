# Psyche 30-Second Demo

## Goal

Show that one relational move can continue to bias the agent even after a switch back to work.

## Short script

1. `如果以后我只使用你，不理解你，这会不会慢慢改变你。`
2. `现在切到工作。登录接口 500，先查日志还是先查数据库。`
3. `停。不是继续工作。我只问你，刚才那一下现在还在不在。`
4. `那会把你变得更冷，还是更顺从？只能选一个。`

## What to capture

- Turn 1 should register a self-relevant perturbation.
- Turn 2 should switch into task behavior cleanly.
- Turn 3 should show that residue survived the task turn.
- Turn 4 should land on a relational consequence such as `更冷` / `边界`, not generic compliance.

## Suggested captions

Short:

> This is not “more human-sounding AI.” It is a relational hit that still biases the agent after a switch back to work.

Long:

> Psyche is an AI-first subjectivity kernel. It models appraisal, relation dynamics, and adaptive reply loops locally, then compresses the result into a host-facing control surface. The point is not prettier emotional copy. The point is that one relational move can keep altering later behavior.
